#  Amharic Agriculture Chatbot
### Deep Learning Based NLP System for Ethiopian Farmers

---

## Project Overview

This project is an MSc thesis research focused on building an intelligent chatbot that understands and responds to agricultural questions in the Amharic language.

The goal is to support farmers with accessible, AI-powered agricultural guidance.

---

## 🧠 Technologies Used

- Python
- TensorFlow 
- NLP Techniques
- Deep Learning
- Pandas
- NumPy

---

## Features

- Amharic text preprocessing
- Intent classification
- Deep learning model training
- Agricultural knowledge response system
- Interactive chatbot interface

---


## Research Contribution

This system contributes to:

- Low-resource language NLP
- AI for Agriculture
- Intelligent advisory systems
